    Hi, I'm Makrand Gurjar 👋

    I am a passionate software developer and technology enthusiast currently focused on building modern web applications and real-world problem-solving solutions.

    Cashmetry is a fully functional expense tracking application designed to simulate a real fintech product. I built this app from scratch using HTML, CSS, and JavaScript with a focus on clean UI, user experience, and practical financial features.

    This application includes income and expense tracking, analytics dashboard, budget planning, savings goals, AI-based financial advice, export reports, feedback system, and multi-user functionality — all working offline without any backend.

    My goal while building Cashmetry was to demonstrate full product thinking, not just coding — including usability, performance, and real-world practicality.

    Technologies Used: HTML5 • CSS3 • JavaScript • Chart.js • jsPDF • SheetJS • LocalStorage • PWA concepts

    Designed and developed independently as a portfolio and product demonstration project.
