# 💰 Cashmetry — Expense Tracker Web App

Cashmetry is a premium fintech-style expense tracking application built to simulate a real-world financial product experience.

## ✨ Features

- Income & Expense Tracking
- Analytics Dashboard
- Budget Planning & Savings Goals
- AI Financial Insights
- Wallet Top-Up Simulation
- Export to PDF & Excel
- Multi-user Local Login
- Feedback System
- Developer Profile Section
- Works Fully Offline

## 🚀 Tech Stack

HTML • CSS • JavaScript • Chart.js • jsPDF • SheetJS • LocalStorage
